# Blueprint — Site Ilumine Representações

## Objetivo
Criar um site institucional simples para validação com cliente, apresentando a Ilumine Representações, suas linhas de produtos e botões de solicitação de orçamento via WhatsApp.

## Público-alvo
- Empresas
- Prefeituras
- Licitações
- Comércios
- Indústrias
- Escritórios
- Postos de combustível

## Estilo visual
- Minimalista, técnico e corporativo
- Cores: branco, preto, cinza e detalhes em azul discreto
- Referência estética: sites de iluminação profissional como Stella Iluminação
- Visual limpo, premium, com foco em produto e orçamento

## Stack recomendada
- HTML
- CSS
- JavaScript
- JSON para catálogo
- GitHub Pages ou Cloudflare Pages

## Páginas
1. Home
2. Sobre
3. Catálogo
4. Contato

## Seções da Home
- Hero principal
- Categorias de produtos
- Destaques técnicos
- Chamada para orçamento
- Footer com contato
